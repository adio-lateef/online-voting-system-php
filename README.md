# 🗳️ Online Voting System (PHP & MySQL)

This project is a web-based voting system developed using PHP and MySQL. It allows users to register, log in, and cast votes securely, while administrators manage candidates and results.

---

## 🚀 Features

- User registration and login  
- Admin authentication system  
- Voting functionality  
- Candidate management  
- Session handling for users  

---

## 🛠️ Technologies Used

- PHP  
- MySQL  
- HTML/CSS  
- XAMPP (Local Server)

---

## 📁 Project Structure
online-voting-system-php/
│
├── index.php
├── login.php
├── reg.php
├── database/
│ └── vote.sql


---

## ⚙️ How to Run

1. Install XAMPP  
2. Move project to:htdocs/

3. Start:
- Apache  
- MySQL  

4. Open phpMyAdmin  
5. Import:database/vote.sql
6. Visit:http://localhost/online-voting-system-php
